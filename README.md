### Portfolio
To run:

1. Clone the repo.
2. Run 
    ```bash
    npm i
    ```
    or
    ```bash
    yarn install
    ```
    To install all dependecies
3.  To run the development server:
    ```bash
    npm run dev
    ```
    or
    ```bash
    yarn dev 
    ```

You can view the project at `https://localhost:5000`.
